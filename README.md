# ZabanPeik 🌍

A simple, fast, and responsive online text translator built with HTML.

## Features

- 🌐 Translate text between multiple languages
- ⚡ Fast translation
- 📱 Responsive design for mobile and desktop
- 🔊 Text-to-Speech support
- 📋 Copy translated text
- 🌙 Dark mode interface

## Live Demo

https://codewave4.github.io/zabanpeik/

## Technologies

- MyMemory Translation API

## How to Use

1. Select the source language.
2. Enter your text.
3. Select the target language.
4. The translation will appear automatically.
5. Use Copy or Speak buttons if needed.
